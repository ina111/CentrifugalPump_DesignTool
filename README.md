遠心ポンプの設計ツール
================
単段の遠心ポンプの設計ツールです。超小型ロケットエンジン用を想定しているものです。
一般産業用の遠心ポンプの設計ではStepanoffの設計法が有名で一般的ですが、
ここでは電業社機械製作所の論文を参考に豊倉の設計法を採用しています。

決めるべき設計パラメータ
----------------
- ポンプ回転数
- 揚程H
- 流量Q
- 比速度Ns
- 羽根出口角beta2
- ポンプ効率

- 出口径D2
